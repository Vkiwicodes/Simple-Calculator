# Simple Calculator
A simple command-line calculator built using Python.

## Features
- Addition
- Substraction
- Multiplication
- Division
- Handles division by zero
- Continuous menu until exit

## How to Run
When the program asks, enter your choice:
- **1** for Addition
- **2** for Substarction
- **3** for Multiplication
- **4** for Division
- **5** to exit

Then enter the first and second number.

The program performs the selected operation and displays the result. It continues running until you choose **5 (Exit)**

## files
- **main.py** - Contains the calculator program with functions for addition, subtraction, multiplication, and division.

## About
This project was created while learning Python fundamentals - functions, loops, conditional statement, and user input
